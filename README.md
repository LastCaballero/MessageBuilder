# Message‑Builder

Ein moderner, lokal laufender Chat‑Simulator zum Erstellen, Sortieren und Verwalten von Nachrichtenverläufen. Die App benötigt keinen Server und speichert alle Daten ausschließlich im Browser.

---

## ✨ Funktionen

### Mehrere Gespräche
- Neue Gespräche anlegen
- Gespräche auswählen und wechseln
- Gespräche löschen
- Automatische Speicherung in `localStorage`

### Nachrichten erstellen
- Nachrichten als **Ich** oder **Partner** senden
- Automatisches Scrollen
- Moderne Chat‑Bubbles mit Animationen
- Themes: **WhatsApp** und **Viber**

### Kontextmenü
- Rechtsklick auf eine Nachricht öffnet ein Menü
- Aktionen:
  - **Kopieren**
  - **Löschen** (über eindeutige IDs, keine doppelten Löschfehler)

### Drag‑and‑Drop
- Nachrichten per Drag‑and‑Drop neu sortieren
- Reihenfolge wird automatisch gespeichert

### Modernes UI
- Glas‑Effekte
- weiche Schatten
- animierte Buttons
- responsives Layout
- Theme‑Switcher
